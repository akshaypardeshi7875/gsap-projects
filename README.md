# 🎬 GSAP Animation Projects

A collection of creative **web animation projects** built using **HTML, CSS, JavaScript, and GSAP (GreenSock Animation Platform)**.
This repository showcases various animation techniques used in modern web development.

---

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* GSAP (GreenSock Animation Library)

---

## ✨ What’s Inside

This repository contains multiple animation projects such as:

* 🎯 Scroll-based animations
* ✍️ Text and typography animations
* 🎞️ Landing page animations
* 🖱️ Interactive UI effects
* 🎨 Smooth transitions and motion effects

---

## 📂 Project Structure

```text
GSAP/
 ├── 01/
 ├── 02/
 ├── 03/
 ├── 09/
 ├── Apple/
 └── Framer_Motion/
```

Each folder represents a separate project with its own HTML, CSS, and JavaScript files.

---

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/akshaypardeshi/gsap-projects.git
```

2. Open any project folder

3. Run `index.html` in browser
   OR use Live Server (recommended)
